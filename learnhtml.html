Semantic HTML

Element Name Description
<main> Main Encapsulates the dominant content within a page
<footer> Footer Element used to contain info such as contact info, site map, page links, etc..
<section> Section Elements that define chapters, headings, or others with the same theme
<article> Article For articles, blogs, comments, and magazines. Helps accessibility for screen readers.
<aside> Aside Use to mark additional info that can enhance another element
<figcaptiong> Figure Caption Typically is placed inside the <figuregt; element when inserting an image
<audiogt; Audio Used to embedd audio content into a document
<videogt; Vidio Embedd video content and then include atributes to alter playback

Attributes

Attribute Element Description
href <a> Specifies the URL of the page the link goes to
src <img> Embed and image in an HTML page
width <img> Specifies the width of the image (in pixels)
height <img> Specifies the height of the image (in pixels)
alt <img> Alternate text for an image if cannot be displayed
style <p> Adds style to an element such as color, font, size, and more
lang <html> Declare the language of the Web page
title <p> Defines some extra information about an element

HTML Tables

Element Name Description
<table> Table Represents information in a two-dimensional table comprised of rows and columns of cells containing data
<thead> Table Head Defines a set of rows defining the head of the columns of the table
<tbody> Table Body Encapsulates a set of table rows indicating that they comprise the body of the table
<tr> Table Row A row of cells in a table
<th> Table Header Defines a cell as header of a group of table cells
<td> Data Cell Defines a cell of a table that contains data
<col> Table Column Defines a column within a table and is used for defining common semantics on all common cells
<colgroup> Column Group Defines a group of columns within a table
<tfoot> Table Foot Defines a set of rows summarizing the columns of the table

CSS Selectors

Syntax Name Description
* Universal Selects all elements
elementname Type Selects all elements that have the given node name
.classname Class Selects all elements that have the given class attribute
#idname ID Selects an element based on the value of its id attribute
[attr] Attribute Selects all elements that have the given attribute
A B Descendant The (space) combinator selects nodes that are descendants of the first element
A > B Child Selects nodes that are direct children of the first element
A + B Adjacent Sibling Matches the second element only if it immediately follows the first element
